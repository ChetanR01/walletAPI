Node.js API

/create-user POST

Parameters:

body:

{

“name”: string,

“email”: string,

“wallet_id”: string,

“password”: string,

“mobile_no”: string(10)

}

Responses

  1. 500: 
{
status:0,
message:"User already exist with this email or mobile no"
}
  1. 200: 
{
status:1,
message:"User Created Successfully! "
}
